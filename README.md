# Mapa de Habilidades – DIO

## Etapa 1 – Inventário de Habilidades
- Organização de processos (Administração)  
- Testes manuais de software  
- Automação de testes com Selenium e Pytest  
- Testes de API com Postman  
- Consultas SQL para validação de dados  
- Uso de Git/GitHub para versionamento  
- Trabalho em equipe com Scrum e Kanban  
- Comunicação clara e objetiva

## Etapa 2 – Cruzamento com Necessidades do Mercado
- **QA Testing** → alta demanda em empresas de tecnologia.  
- **Automação de testes** → valorizado em times ágeis e projetos digitais.  
- **SQL e Postman** → muito requisitados em vagas de QA e analistas de testes.  
- **Administração e organização** → diferencial para gestão de processos e qualidade.  

---

## Etapa 3 – Caminhos de Monetização
